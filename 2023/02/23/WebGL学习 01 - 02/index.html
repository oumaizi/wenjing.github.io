什么是WebGL

  • a standard for using openGL in browser
  • Use GPU to render

  • HTML Canvas Element
  • WebGL Rendering Context
  • Two Compiled Shaders
    • Vertex Shader
    • Fragment Shader

基本的调用方式

// Vertex 顶点着色器
const vertexShaderSource = `#version 300 es
#pragma vscode_glsllint_stage: vert

void main() {
    gl_Position = vec4(0.0, 0.0, 0.0, 1.0);
    gl_PointSize = 1000.0;
}
`;

// Fragment 碎片着色器
const fragmentShaderSource = `#version 300 es
#pragma vscode_glsllint_stage: frag

precision mediump float;
out vec4 fragColor;

void main() {
    fragColor = vec4(1.0, 0.0, 0.0, 1.0);

}
`;

// Get a WebGL context
const canvas = document.querySelector("#canvas");
const gl = canvas.getContext("webgl2");

// Create and link the shader program
const program = gl.createProgram();
if (!program) {
    throw new Error("Unable to create WebGL program.");
}

// Create a vertex shader object and compile the vertex shader source code
const vertexShader = gl.createShader(gl.VERTEX_SHADER);
gl.shaderSource(vertexShader, vertexShaderSource);
gl.compileShader(vertexShader);

// Attach the vertex shader to the program
gl.attachShader(program, vertexShader);

const fragmentShader = gl.createShader(gl.FRAGMENT_SHADER);
gl.shaderSource(fragmentShader, fragmentShaderSource);
gl.compileShader(fragmentShader);
gl.attachShader(program, fragmentShader);

gl.linkProgram(program);
if (!gl.getProgramParameter(program, gl.LINK_STATUS)) {
    console.log("Could not initialise shaders");
    console.log(gl.getShaderInfoLog(vertexShader));
    console.log(gl.getShaderInfoLog(fragmentShader));
}

// Use the program
gl.useProgram(program);

// Draw
gl.drawArrays(gl.POINTS, 0, 1);

Precision(精度) Qualifier

  • precision highp float
    • float does not have a default precision settings where
    • ![[CleanShot 2023-03-02 at 04.34.11.png]]
  • highp - 24-bit, floating point range: −2126 to 2127, signed integer range: −231 to 231−1, unsigned integer range: 0 to 232−1
  • mediump - 10 bit, floating point range: −214 to 214, signed integer range: −215 to 215−1, unsigned integer range 0 to 216−1
  • lowp - 8/9 bit (signed/unsigned), floating point range: −2 to 2, signed integer range: −28 to 28−1, unsigned integer range 0 to 29−1

Uniform - Global Variables

  • Defined in shaders
  • set with JavaScript
  • sent to vertex shaders and fragment shaders
  • cannot change during a draw

Locations

  • help connect your JavaScript to your WebGL program
  • pointers(指针) faster than names
    • a pointer is a variable that stores the memory address of another variable or object in a computer’s memory.
  • Get locations in WebGL
    • getAttribLocation() returns a Number
    • getUniformLocation() returns an Object
  • ![[CleanShot 2023-03-02 at 14.39.49.png]]
  • gl.uniform4f vs gl.uniform4fv
    • gl.uniform4fv pass an array to the shader
    • WebGL 2: Uniforms (Part 1) - YouTube 
      const fragmentShaderSource = `#version 300 es
#pragma vscode_glsllint_stage: frag
precision mediump float;

uniform int index;
uniform vec4 uColor[3];
out vec4 fragColor;

void main() {
    fragColor = uColor[index];
}
`;

const uColorLocation = gl.getUniformLocation(program, "uColor");
const uIndex = gl.getUniformLocation(program, "index");
gl.uniform1i(uIndex, 1);
gl.uniform4fv(

);

Attributes

  • Attributes are used to define characteristics of a vertex or primitive, such as its position, color, normal direction, and texture coordinates.
  • attribute vs uniform
    • attributes describe the properties of individual vertices or primitives, while uniforms describe the global properties of a shader program.
    • uniforms
      • are available in vertex and fragment shaders
      • cannot be change during a draw call
    • attributes
      • are available only in vertex shaders
      • changes fro every vertex in the buffer
  • syntax
    • WebGL 1: attribute vec4 aPos
    • WebGL 2: in vec4 aPos
  • Limitations
    • each WebGLProgram is guaranteed as least 16 attributes
    • gl.getParameter(gl.MAX_VERTEX_ATTRIBS) to get limitations
  • Locations
    • gl.getAttribLocation() return a Number
    • do not change over the life of the program
    • can set an attribute location
    • gl.bindAttribLocation(program, 0, "aAttributeName")

Buffer

  • In WebGL, a buffer is a region of memory on the GPU that stores a collection of data, such as vertices, colors, or texture coordinates. Buffers are used to transfer data from the CPU to the GPU, where it can be processed by the graphics pipeline.
